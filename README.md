# First AI Project 🤖

This is my first experiment exploring Machine Learning and Artificial Intelligence using **Node.js** and **TensorFlow.js**. The goal of this project is to understand the fundamentals of neural networks and data processing in a backend environment.

---

## 🚀 Technologies

* **Node.js** - JavaScript runtime environment.
* **TensorFlow.js** (`@tensorflow/tfjs-node`) - A library for training and deploying ML models in Node.js.
